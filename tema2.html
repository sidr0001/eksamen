Tema 02

Grundlæggende web

Introduktion

Vi blev introduceret til de grundlæggende elementer i webudvikling, herunder HTML og CSS, samt hvordan man strukturerer og placerer indhold og billeder ved hjælp af grid.

Fremgangsmåde og processer

I dette tema lavede vi studiestartsprøven, hvor vi med udgangspunkt i layoutdiagrammer og wireframes skulle udvikle et responsivt website og for første gang afprøve vores kodeevner i praksis.Opgaven havde fokus på at strukturere indholdet semantisk med HTML og style det med CSS ved hjælp af grid, flexbox, margin, padding og media queries.

Derudover arbejdede vi med farver og typografi, som skulle understøtte både sidens æstetik og brugervenlighed. Vi blev introduceret til centrale designkonventioner og gestaltprincipper - herunder lovene om nærhed, lighed, kontinuitet og lukkethed.Disse principper gav os en dybere forståelse for, hvordan visuelle elementer opfattes, og hvordan man kan skabe mere intuitive og brugervenlige designs.

Hvad har jeg lært på dette tema?

Gennem dette tema har jeg opnået en styrket forståelse for de mange overvejelser og faktorer, der indgår i designet af et velfungerende website.Jeg har lært at anvende grundlæggende værktøjer inden for HTML og CSS, hvilket har gjort mig i stand til at udvikle responsive og brugervenlige designs. Derudover har jeg opnået erfaring med billedbehandling, så billederne optimeres til web - både i forhold til filstørrelse og visuel kvalitet.
